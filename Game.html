Adsorption Game

This simple game describes adsorption in terms of the impurity model. For pedagogical purposes, the impurity model is introduced with the help of two well known for chemists model systems, namely, the hydrogen atom and the Hückel method. The adsorption is a complex process that involves many steps. In order to keep things simple, we consider here only (1) the interaction of a benzene molecule with a hydrogen atom, which is placed above the benzene ring along the C6 axis of symmetry; (2) the interaction of a naphthalene molecule with a hydrogen atom, which is placed above naphthalene along the C2 axis of symmetry. The resulting impurity model is constructed similar to the procedure of the ligand field theory, i.e. the π-orbitals of benzene or naphthalene are considered as the ligand orbitals of the hydrogen atom. Then, the ground state energy is calculated in terms of the configuration interaction method. This simple model can be extended to demonstrate the hydrogen adsorption on graphene fragments by extending the number of the fused benzene rings.

Objective

The main goal of this game is to introduce the configuration interaction method in the form of a game as well as to present the concepts of

  • molecular orbitals

  • electronic configurations

  • molecular symmetry

  • a variational principle

in a playful form.

Prerequisites

Although this program was developed for undergraduate chemistry students, anybody can play Adsorption Game. However, it is desirable to be familiar with the above mentioned concepts in order to develop the winning strategy. A short introductory course into the concepts used in the game can be found here.

Model description

A simple impurity model used in this game contains only one correlated orbital. Namely, it is the hydrogen atom orbital. It means that the two-electron operator is invoked only for the doubly occupied hydrogen orbital (Hubbard operator). Estimations for the numerical values of the Hückel π-orbitals are taken from the DFT calculations. The process described by this model involves a chemical reaction between the benzene molecule and the adsorbate (chemisorption - a kind of adsorption). Such changes imply the change in the electronic structure. However, the molecular orbitals are not optimized for the combined hydrogen-benzene system. Instead, the configuration interaction is employed. The matrix elements between two different configurations are calculated in this model with the fully symmetric one-electron operator involving one hydrogen and one π-orbital. Other details will be given elsewhere.

How to play

You have to get as close as possible to the precalculated ground state energy by selecting as many configurations as needed from the Top Left window. In order to increase the relative weight of a particular configuration you have to enter it again. It will change the trial wave function and the corresponding energy.

You have to play quite fast, because every 20 seconds the current configuration will be added to your trial wave function automatically. To increase your chances to win, you can preview all configurations with their energies in the Top Left window before you press Start button. It is advised to study this information before you play. Each configuration consists of a set of orbitals. The red circle in the Top Left window is connected to the orbitals shown in the Bottom Right window. It is also possible to preview all orbitals before the game begins.

The molecular orbitals are given as a hint to help you to find the non zero matrix elements between two different configurations. Note that the irreducible representation labels for the π-orbitals of benzene are given for the D6h point group, and the π-orbitals of naphthalene are described in term of the C2h point group. You have to know the corresponding irreducible representations for the symmetry of the hydrogen-benzene or hydrogen-naphthalene complex.

Model 2 game (hydrogen-naphthalene complex) has been introduced as a next level. It is more difficult, because differen configurations in the wave function can have opposite signs. Therefore, an additional option has been added for the second model - entering configurations with a negative sign ("Subtract" button). Thus, you should become familiar with the Moldel 1, before you decide to switch to the higher level.

You can preview your results in the Bottom Left window switching between the energy and wave function modes. In the last case the weights of the configurations are shown. It is possible to preview all configurations added to your trial wave function in the Bottom Middle window by clicking on the intensity peaks. The matrix element field will show the value of the matrix element between two configurations shown in the Top Left and Bottom Middle windows when the game begins. The number of steps is not limited in the current version of the program, but you can stop the game any time and review your results.



How to use Help mode

If you could not obtain the lowest energy after hundreds of steps you can use the help mode to refine your strategy. The help mode contains hints in the Top Left window, explaining how each configuration is related to the initial one. For example, it says whether it is a single or double excitation from the initial configuration. Comparing this information to the model description you can estimate which matrix element is non vanishing. In order to do this you need to use the symmetry information from the Bottom Right window. In the help mode this window shows the irreducible representation labels for the orbitals in the C6v point group. The last step is to use this information to calculate the symmetry of the product of the irreducible representations for a particular matrix element. All necessary information can be found in the product table for the C6v point group below.

Product table for C6v point group
A1 A2 B1 B2 E1 E2
A1 A1 A2 B1 B2 E1 E2
A2 A2 A1 B2 B1 E1 E2
B1 B1 B2 A1 A2 E2 E1
B2 B2 B1 A2 A1 E2 E1
E1 E1 E1 E2 E2 A1+A2+E2 B1+B2+E1
E2 E2 E2 E1 E1 B1+B2+E1 A1+A2+E2

You have to know the criteria for the molecular integrals to be nonzero to make use of this table in the game.

What is your score?

The number of steps taken to obtain the ground state energy is your score in this game. The fewer steps you have taken the better.
